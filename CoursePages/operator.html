JavaScript Operators

Learn about the various operators available in JavaScript.

Arithmetic Operators

Arithmetic operators are used to perform arithmetic on numbers.

  • Addition (+): Adds two numbers, e.g., 5 + 2 results in 7.
  • Subtraction (−): Subtracts the second number from the first, e.g., 5 - 2 results in 3.
  • Multiplication (*): Multiplies two numbers, e.g., 5 * 2 results in 10.
  • Division (/): Divides the first number by the second, e.g., 10 / 2 results in 5.
  • Modulus (%): Returns the remainder of a division, e.g., 10 % 3 results in 1.
  • Increment (++): Increases a variable by 1, e.g., i++.
  • Decrement (−−): Decreases a variable by 1, e.g., i--.

Assignment Operators

Assignment operators assign values to variables.

  • Assignment (=): Assigns a value to a variable, e.g., x = 10.
  • Addition assignment (+=): Adds a value to a variable, e.g., x += 5 is equivalent to x = x + 5.
  • Subtraction assignment (-=): Subtracts a value from a variable, e.g., x -= 5 is equivalent to x = x - 5.
  • Multiplication assignment (*=): Multiplies a variable by a value, e.g., x *= 5 is equivalent to x = x * 5.
  • Division assignment (/=): Divides a variable by a value, e.g., x /= 5 is equivalent to x = x / 5.
  • Modulus assignment (%=): Assigns the remainder of the division to a variable, e.g., x %= 5 is equivalent to x = x % 5.

Comparison Operators

Comparison operators compare two values and return a boolean result.

  • Equal to (==): Checks if two values are equal, e.g., 5 == '5' results in true.
  • Equal value and type (===): Checks if two values and types are equal, e.g., 5 === '5' results in false.
  • Not equal (!=): Checks if two values are not equal, e.g., 5 != '5' results in false.
  • Not equal value or type (!==): Checks if two values and types are not equal, e.g., 5 !== '5' results in true.
  • Greater than (>): Checks if the first value is greater than the second, e.g., 10 > 5 results in true.
  • Less than (<): Checks if the first value is less than the second, e.g., 5 < 10 results in true.
  • Greater than or equal to (>=): Checks if the first value is greater than or equal to the second, e.g., 10 >= 10 results in true.
  • Less than or equal to (<=): Checks if the first value is less than or equal to the second, e.g., 5 <= 10 results in true.

Logical Operators

Logical operators are used to combine multiple conditions.

  • Logical AND (&&): Returns true if both conditions are true, e.g., true && false results in false.
  • Logical OR (||): Returns true if at least one condition is true, e.g., true || false results in true.
  • Logical NOT (!): Reverses the boolean value, e.g., !true results in false.

Example

Here’s an example demonstrating the use of different operators:


// Arithmetic operators
let x = 10;
let y = 5;
console.log(x + y); // 15
console.log(x - y); // 5
console.log(x * y); // 50
console.log(x / y); // 2
console.log(x % y); // 0

// Assignment operators
let a = 10;
a += 5;
console.log(a); // 15

// Comparison operators
console.log(10 == "10"); // true
console.log(10 === "10"); // false
console.log(10 != "10"); // false
console.log(10 !== "10"); // true

// Logical operators
let condition1 = true;
let condition2 = false;
console.log(condition1 && condition2); // false
console.log(condition1 || condition2); // true
console.log(!condition1); // false
