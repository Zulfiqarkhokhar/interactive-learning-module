JavaScript DOM Manipulation

Learn how to manipulate the Document Object Model (DOM) using JavaScript.

Introduction to DOM

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content. DOM manipulation refers to the act of changing the content or structure of a webpage using JavaScript.

Selecting Elements

To manipulate the DOM, you first need to select the elements you want to interact with. This can be done using methods like getElementById, getElementsByClassName, getElementsByTagName, and querySelector/querySelectorAll.

Changing Content

Once you have selected the elements, you can change their content using properties like innerHTML, textContent, or innerText.

Changing Styles

You can also change the appearance of elements by modifying their CSS styles using the style property or by adding/removing CSS classes using the classList property.

// Selecting an Element
let titleElement = document.getElementById("title");
console.log(titleElement.textContent); // Original Title

// Changing Content
titleElement.textContent = "Updated Title";
console.log(titleElement.textContent); // Updated Title

// Changing Styles
titleElement.style.color = "blue";
titleElement.style.fontSize = "2em";

// Adding and Removing Classes
titleElement.classList.add("highlight");
titleElement.classList.remove("highlight");

// Creating and Appending New Elements
let newParagraph = document.createElement("p");
newParagraph.textContent = "This is a new paragraph.";
document.body.appendChild(newParagraph);

// Removing Elements
let oldParagraph = document.getElementById("oldParagraph");
oldParagraph.remove();
