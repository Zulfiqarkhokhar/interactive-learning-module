JavaScript Objects and Arrays

Learn about JavaScript objects and arrays, two fundamental data structures in the language.

Objects

Objects are collections of key-value pairs, where each key is a string (also called a property), and the value can be of any type. They are used to store related data and functions (methods) together.

Creating Objects

There are several ways to create objects in JavaScript, including using object literals and constructors.

Accessing and Modifying Properties

Properties of an object can be accessed and modified using dot notation or bracket notation.

Arrays

Arrays are ordered collections of values, which can be of any type. Each value in an array is called an element, and each element has a numbered position, known as its index.

Common Array Methods

Arrays come with various built-in methods for common tasks such as adding, removing, and iterating over elements.

// Creating an Object
let person = {
    name: "John",
    age: 30,
    greet: function() {
        return "Hello, " + this.name;
    }
};
console.log(person.name); // John
console.log(person.greet()); // Hello, John

// Adding and Modifying Properties
person.job = "Developer";
person.age = 31;
console.log(person.job); // Developer
console.log(person.age); // 31

// Creating an Array
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[0]); // apple

// Adding and Removing Elements
fruits.push("orange"); // Add to end
fruits.pop(); // Remove last element
fruits.unshift("mango"); // Add to start
fruits.shift(); // Remove first element
console.log(fruits); // ["apple", "banana", "cherry"]

// Iterating Over Arrays
fruits.forEach(function(fruit, index) {
    console.log(index + ": " + fruit);
});

// Array Methods
let numbers = [1, 2, 3, 4, 5];
let doubled = numbers.map(function(number) {
    return number * 2;
});
console.log(doubled); // [2, 4, 6, 8, 10]
